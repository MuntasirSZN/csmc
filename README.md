## Tech Stack

- Next.js
- Turso (libSQL Database)
- Drizzle ORM
- Tailwind CSS
- React 19
- Shadcn UI
- Eslint
- Typescript 
- Netlify
- Better Auth
- Upstash Redis
- Bun

## Developing

It uses app router of next.js, with [src](./src) directory.

1. Clone this repo.

```sh
git clone https://github.com/MuntasirSZN/csmc
```

2. `cd` to that directory and run:

```sh
bun setup
```
